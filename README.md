# 🏖️ Gerenciamento de Férias - MVP

Este projeto é um MVP (Produto Mínimo Viável) desenvolvido para a disciplina de **Projeto de Bloco: Desenvolvimento Back-end**. O sistema automatiza o fluxo de solicitações de férias entre Colaboradores, Gestores e RH.

## 👤 Autor
* **Murilo Augusto Garmatter Gonsalves**

---

## 🛠️ Tecnologias
* **Linguagem:** C# (.NET 8)
* **Framework:** ASP.NET Core Razor Pages
* **Banco de Dados:** Entity Framework Core (In-Memory)
* **Interface:** Bootstrap 5

---

## 📐 Sobre o Projeto
O desenvolvimento foi guiado por processos de engenharia de software, incluindo a criação de Diagramas UML (Casos de Uso, Classes, Sequência e Estados) que serviram de base para a arquitetura do código.

### Estrutura Principal:
* `/Models`: Entidades e Regras de Negócio.
* `/Pages`: Interfaces divididas por perfis de acesso.
* `/Data`: Configuração do banco de dados em memória.

---

## 💻 Como Rodar
1. Clone o repositório: <br>
```git clone https://github.com/MuriloGarmatter/Gerenciamento-Ferias-ASPNet.git ``` <br>
2. Acesse a pasta do Projeto: <br>
```cd Gerenciamento-Ferias-ASPNet``` <br>
3. Restaure as dependências: <br>
```dotnet restore``` <br>
4. Execute a aplicação: <br>
```dotnet run``` <br>
5. Após rodar o projeto, abra o navegador e acesse: <br>
```https://localhost:xxxx``` <br>
A porta será exibida no terminal ao iniciar a aplicação.
